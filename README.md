# Movie-ticket-
Booking system designed for booking movie tickets.
USING C++ AND OOP CONCEPTS

PROJECT PURPOSE:
  The purpose of this project is to develop a simple Movie Ticket Booking System using C++
  and Queue data structure. The project helps in understanding queue operations and basic
  Object-Oriented Programming concepts by managing movie ticket bookings, cancellations,
  and customer details in an organized way.
  
CONCEPTS USED:
   Class
   Object
   Encapsulation
   Member functions
   Abstraction
   Data container: Queue
   
QUEUE OPERATIONS USED:
   Push () → Add booking
   Pop () → Cancel booking
   Front () → Show first booking
   Back () → Show latest booking
   Size () → Total bookings
   Empty () → Check queue empty or not
   
ALGORITHM:
1. Start the program.
2. Display menu options.
3. Take user choice.
4. If booking selected:
   Enter customer details.
   Select movie, language, screen type, timing, and seats.
   Calculate total amount.
   Add booking to queue.
5. Perform queue operations based on user choice.
6. Repeat until user exits.
7. End program.
   
FEATURES PROVIDED IN THE PROGRAM:
  Movie selection
  Language selection
  Screen type selection (IMAX, 3D, 2D)
  Show timing selection
  Seat booking
  Automatic ticket price calculation
  Housefull condition when queue is full

CONCLUSION:
The program successfully demonstrates the use of Queue and OOP concepts in C++ by
creating a simple Movie Ticket Booking System with booking and cancellation features.
